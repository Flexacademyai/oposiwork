Politica de privacidad

Ultima actualizacion: 20 de mayo de 2026.

1. Responsable del tratamiento

Responsable: Oposiwork.

Contacto de privacidad y ejercicio de derechos: soporte@oposiwork.com.

El usuario puede contactar con Oposiwork para cualquier consulta sobre privacidad, seguridad, soporte o ejercicio de derechos relacionados con sus datos personales.

2. Datos que tratamos

Podemos tratar datos de cuenta (email, nombre y apellidos), autenticacion, preferencias, oposiciones seguidas, progreso de estudio, resultados de ejercicios, alarmas, notificaciones, consentimiento de voz/IA, datos tecnicos de seguridad y datos de suscripcion.

3. Finalidades

  • Crear y mantener la cuenta de usuario.
  • Mostrar oposiciones, convocatorias, contenido de estudio y progreso.
  • Gestionar suscripciones, pagos web con Stripe y pagos moviles con las tiendas correspondientes.
  • Controlar el acceso premium y la descarga unica de PDFs.
  • Enviar notificaciones de convocatorias, recordatorios y comunicaciones operativas.
  • Prevenir abuso, fraude, accesos no autorizados y usos maliciosos.
  • Atender solicitudes de soporte y ejercicio de derechos.

4. Base juridica

El tratamiento se basa, segun el caso, en la ejecucion del contrato o medidas precontractuales, el consentimiento del usuario, el cumplimiento de obligaciones legales y el interes legitimo en mantener la seguridad de la plataforma.

5. Proveedores y destinatarios

Oposiwork usa proveedores necesarios para prestar el servicio, entre ellos Supabase (base de datos, autenticacion y storage), Vercel (hosting), Stripe (pagos web), RevenueCat (gestion de suscripciones), Firebase/Google (notificaciones push cuando se activen), Resend (email transaccional cuando se active) y proveedores de IA para funciones premium cuando el usuario las solicite.

No vendemos datos personales.

6. Transferencias internacionales

Algunos proveedores pueden tratar datos fuera del Espacio Economico Europeo. En ese caso se deben usar garantias adecuadas, como decisiones de adecuacion, clausulas contractuales tipo u otros mecanismos previstos por la normativa aplicable.

7. Conservacion

Los datos se conservan mientras la cuenta este activa, mientras sean necesarios para prestar el servicio o durante los plazos exigidos por obligaciones legales, fiscales, contables o de seguridad. El usuario puede solicitar la supresion cuando proceda.

8. Derechos

El usuario puede solicitar acceso, rectificacion, supresion, oposicion, limitacion del tratamiento y portabilidad escribiendo a soporte@oposiwork.com. Tambien puede presentar una reclamacion ante la Agencia Espanola de Proteccion de Datos.

9. Seguridad

La plataforma aplica Row Level Security en Supabase, separacion de claves por entorno, validacion de entradas, URLs firmadas de corta duracion para descargas, CSP, HSTS y funciones de servidor para operaciones sensibles.

10. Menores

Oposiwork no esta dirigida a menores de 14 anos. Si detectamos una cuenta creada por un menor sin base valida, podremos bloquearla o eliminarla.
